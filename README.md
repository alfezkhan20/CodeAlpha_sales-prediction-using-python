# Sales Prediction using Python 📊

## 📋 Project Overview
This project focuses on predicting future sales based on advertising expenditure across TV, Radio, and Newspaper platforms. Using **Machine Learning (Linear Regression)**, we analyze the correlation between advertising budgets and total sales.

## 🛠️ Technologies Used
- **Environment:** Google Colab / Jupyter Notebook
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

## 📂 Repository Structure
- `sales_prediction.ipynb`: The complete Python notebook with data analysis and model building.
- `Advertising.csv`: The dataset used for training and testing.
- `requirements.txt`: Necessary libraries to run the project locally.

## 📈 Key Achievements
- Performed Exploratory Data Analysis (EDA) to find sales trends.
- Built a predictive model using **Linear Regression**.
- Achieved an accurate sales forecast based on marketing spend.
